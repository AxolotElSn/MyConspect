1) git init - начало слежения за файлами
2) git config --local user.name "Name"	(если скачали гит впервые на пк то надо поменять local на global)
  git config --local user.email blabla@bla.com 
3) git status - статус, как ни странно)
4) git add -A добавление ВСЕХ файлов в версию
5) git commit -a -m"first commit" контрольная точка 
6) копируем команду с репозитория гитхаба
7) git push - отправка на гит
-------------------------------------------------------------------------------------------------------------------------------
если надо скопировать проект с гита
1) git clone + ссылка с репозитория гита + имя папки в которую нужно скопировать
если мы склонировали проект с гита, то и пушиться от нас он будет на этот же гит
2) git pull добавляет изменения с гита в проекте (Обязательно прописать перед работой)


compact folders - убрать галочку чтоб ебучие папки открывались нормально
npm install --legacy-peer-deps - команда если npm i выдает ошибку с конфликтом версий
